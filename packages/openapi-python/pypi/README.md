# Hey API

OpenAPI to Python code generator. Production-grade SDKs and Pydantic models. Built by [Hey API](https://heyapi.dev).

## Installation

### uv

Install:

```sh
uv add hey-api
```

Or run without installing:

```sh
uvx hey-api --help
```

### pip

Install:

```sh
pip install hey-api
```

Or run without installing:

```sh
pipx run hey-api
```

## Usage

```sh
hey-api --help
```
