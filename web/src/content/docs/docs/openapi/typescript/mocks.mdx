# Mocks

Realistic mock data is an important component of every robust development process, testing strategy, and product presentation.

## Options

Hey API natively supports the following mocking frameworks.

- [Chance](/openapi-ts/plugins/chance) <Badge text="Vote" variant="tip" />
- [Faker](/openapi-ts/plugins/faker) <Badge text="Vote" variant="tip" />
- [Falso](/openapi-ts/plugins/falso) <Badge text="Vote" variant="tip" />
- [MSW](/openapi-ts/plugins/msw) <Badge text="Vote" variant="tip" />
- [Nock](/openapi-ts/plugins/nock) <Badge text="Vote" variant="tip" />
- [Supertest](/openapi-ts/plugins/supertest) <Badge text="Vote" variant="tip" />

Don't see your framework? Let us know your interest by [opening an issue](https://github.com/hey-api/openapi-ts/issues).
