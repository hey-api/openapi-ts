# Mocks

Realistic mock data is an important component of every robust development process, testing strategy, and product presentation.

## Plugins

Hey API natively supports the following mocking frameworks.

- [Chance](/docs/openapi/typescript/plugins/chance) <Badge text="Vote" variant="tip" />
- [Faker](/docs/openapi/typescript/plugins/faker) <Badge text="Vote" variant="tip" />
- [Falso](/docs/openapi/typescript/plugins/falso) <Badge text="Vote" variant="tip" />
- [MSW](/docs/openapi/typescript/plugins/msw) <Badge text="Vote" variant="tip" />
- [Nock](/docs/openapi/typescript/plugins/nock) <Badge text="Vote" variant="tip" />
- [Supertest](/docs/openapi/typescript/plugins/supertest) <Badge text="Vote" variant="tip" />

Don't see your framework? Let us know by [opening an issue](https://github.com/hey-api/openapi-ts/issues).
