# Migrating

Hey API is in active development, which allows us to move quickly and continuously improve the project. That pace sometimes comes with breaking changes. This page documents those changes and provides migration guidance for each release. If you need help with a migration, please [open an issue](https://github.com/hey-api/hey-api/issues).
